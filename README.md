<a name="english"></a>
## :gb: English

### Overview

A professional system for generating e-learning content in SCORM 1.2 format using an **AI multi-agent architecture**. Each agent handles a specialized phase: content creation, formatting, multimedia integration, and SCORM packaging.

### Features

- **Multi-Agent AI** - Specialized agents for each creation phase
- **SCORM 1.2 Compliant** - Works with any LMS (Moodle, Canvas, Blackboard, etc.)
- **Markdown to HTML** - Automatic conversion with professional styling
- **Multimedia Support** - Audio/video players with completion tracking
- **Progress Tracking** - Full SCORM API communication with LMS

### Supported LMS

Moodle, Canvas, Blackboard, Google Classroom, TalentLMS, Schoology, D2L Brightspace, SAP Litmos, Docebo, iSpring Learn, and any SCORM 1.2 compatible system.

### Why SCORM 1.2?

| Advantage | Description |
|-----------|-------------|
| Universal Compatibility | Works on any modern LMS |
| Reliability | Mature standard since 2001 |
| Tracking | Completion and progress support |
| Portability | Single package works everywhere |
| Offline-Ready | Viewable locally without LMS |

### Quick Start

```bash
# Clone
git clone https://github.com/fracabu/scrom-course-generator.git
cd scrom-course-generator

# Install
pip install -r requirements.txt

# Generate course (with Claude Code)
# Use the multi-agent workflow to create SCORM packages
```

### Requirements

- Python 3.8+
- Claude Code (for AI agents)
- ffmpeg (optional, for media compression)
